# RbxUp

Адаптивный демонстрационный интернет-магазин игровой валюты Robux. Проект не принимает платежи, не запрашивает пароли или банковские данные и не связан с Roblox Corporation.

## Структура

- `index.html` — главная страница;
- `catalog.html` — каталог из семи наборов;
- `css/style.css` — общие стили и адаптивность;
- `js/main.js` — мобильное меню, раскрытие гайдов и анимации;
- `js/cart.js` — каталог, корзина, `localStorage` и демо-оформление;
- `assets/` — локальная иллюстрация и favicon.

## Локальный запуск

Сайт можно открыть двойным кликом по `index.html`. Для проверки в условиях, близких к GitHub Pages, лучше запустить локальный сервер из корня проекта:

```bash
python3 -m http.server 8000
```

Затем открыть `http://localhost:8000/`.

## Публикация на GitHub Pages

1. Создайте репозиторий на GitHub и загрузите в него содержимое этой папки.
2. Откройте **Settings → Pages**.
3. В блоке **Build and deployment** выберите **Deploy from a branch**.
4. Выберите ветку `main`, папку `/ (root)` и нажмите **Save**.
5. После публикации ссылка появится в том же разделе.

Дополнительный workflow не нужен: проект статический, а все пути относительные.

## Возможности

- две связанные страницы и мобильное меню;
- адаптивность от 320 пикселей;
- добавление, изменение количества, удаление и очистка корзины;
- автоматический пересчёт количества и суммы;
- сохранение корзины между перезагрузками;
- доступная боковая панель, управление кнопками с клавиатуры и закрытие по `Escape`;
- демонстрационное окно оформления без настоящей оплаты;
- поддержка `prefers-reduced-motion`.
